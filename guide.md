## ğŸ› Troubleshooting

**Can't build houses?**
- Check if you own the complete color set (monopoly)
- Check if you have enough money
- Check the even building rule - you might need to build on other properties first

**Can't mortgage?**
- Properties must have NO buildings to be mortgaged
- Sell all buildings first using Sell Mode

**Why can't I sell a specific building?**
- Even building rule: You can only sell from properties with the MOST buildings in that color group

**Trade not working?**
- Make sure you've selected a trading partner first
- AI players cannot initiate or receive trades
